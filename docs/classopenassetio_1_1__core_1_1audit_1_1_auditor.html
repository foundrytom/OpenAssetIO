Detailed Description

This class provides a quick-and-dirty accounting mechanism for Class, method and object usage.

The idea is to look at the extents of usage, rather than any kind of realtime reporting.

Raw coverage data is accessible, or can be sprinted to a string.

def coverage (   self)
Returns
dict, The main coverage data dict with all data since the last reset. It is a hierarchical dictionary where at any level two keys represent the coverage data: kKey_Count (int) and kKey_Args (list). Other keys in the dict represent child counts. For example top-level keys are Classes or arbitrary objects. Other keys under a Class dict are the methods of that Class.
