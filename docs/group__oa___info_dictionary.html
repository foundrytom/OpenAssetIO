Enumeration Type Documentation

enum oa_InfoDictionary_ValueType

Enumeration of the available types in a InfoDictionary.

The set of possible types is dictated by those specified in the definition of the variant value type. In particular, this means the set of types is fixed and cannot be extended by hosts or plugins. This enum is therefore exhaustive.
