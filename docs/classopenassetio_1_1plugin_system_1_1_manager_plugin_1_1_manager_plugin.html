Detailed Description

This class represents the various derived classes that make up the binding to a Asset Management System.

It used by the dynamic plug-in discovery mechanism (openassetio.pluginSystem.PluginSystem) to instantiate the main classes in an implementation.

The class will never be instantiated itself, so all functionality is via class methods.

In order to register a new asset management system, simply place a python package on the appropriate search path, that has a top-level attribute called 'plugin', that holds a class derived from this.

Warning
This class, may be used in a batch, or UI session, so consequently, it is imperative that no ui libraries (QtCore, QtGui etc...) are imported unless uiDelegate() is called, and ideally, even then, this should be deferred until something is requested from the openassetio-ui.implementation.ManagerUIDelegate.

def interface (   cls)

Constructs an instance of the openassetio.managerApi.ManagerInterface.

This is an instance of some class derived from ManagerInterface to be bound to the Host-facing openassetio.hostApi.Manager.

Generally this is only directly called by the openassetio.pluginSystem.PluginSystemManagerFactory. It may be called multiple times in a session, but there as the ManagerInterface API itself is specified as being stateless (aside from any internal caching/etc...) then there is no pre-requisite to always return a new instance.
