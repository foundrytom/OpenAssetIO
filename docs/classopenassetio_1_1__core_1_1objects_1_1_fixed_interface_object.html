Detailed Description

This class is a simple extension to object, to dis-allow get/set of any attributes not defined with the class.

This can be useful to make introspectable objects with meaningful help() messages by creating data members like so:

/code numCakes = TypedProperty(int, doc="Some member variable") edible = UntypedProperty() /endcode
