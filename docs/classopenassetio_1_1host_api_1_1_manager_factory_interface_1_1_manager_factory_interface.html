Detailed Description

Manager Factories are responsible for instantiating classes that derive from openassetio.managerApi.ManagerInterface or openassetio-ui.implementation.ManagerUIDelegate for use within an host.

ManagerFactoryInterface defines the abstract interface that any such factory must adopt.
